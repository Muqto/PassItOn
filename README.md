### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Muqto/Photo-Share-client.git
   ```
2. cd into the new folder and install NPM packages
   ```sh
   npm install
   ```
3. To run on IOS
    ```sh
   npm run ios
   ``` 
3. To run on Android, download android studio and open an emulator for an android phone, then run
    ```sh
   npm run android
   ```
